# ⚙️ INSTALLATION & SETUP

Clone the repository:

```bash
git clone https://github.com/martinmounir/My_Portfolio.git
```

Go to the project directory:

```bash
cd My_Portfolio
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```
